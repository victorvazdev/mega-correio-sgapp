# Mega Correio - Wireframes, Canvas, Backlog e Sprint
Mega Correio é foi inspirado em um clássico problema entre pessoas com menos afinidade em tecnologia, o momento de comunicar o e-mail ou até mesmo lembra-lo.

Ele é um provedor de e-mail completo, com acesso quanto web quanto móvel, focado 100% no público brasileiro com o intuito de facilitar a memorização e compartilhamento de endereço de e-mails.

Esse repositório contém o Wireframe do projeto móvel, o Canvas MVP e Backlog juntamente com a Sprint 1 feito no Jira.
